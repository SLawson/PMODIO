Writes data to LCD at the current position


Parameters

  

    
inst_pPMODIO instance address 

    
datadata to write 

    
maskmask to select which bits are written

  

  




Returns
XST_SUCESS


Note
The PMODIO instance's current X position will be automatically incremented by 1 on every write.



Since writes to the LCD must be done a full byte at a time, the mask argument is provided to prevent the caller from having to perform read-modify-write cycles when a data write overlaps with a previous write. The driver tracks all writes internally, and when writing new data, it handles the read-modify-write cyles to only write bits to the LCD that are enabled by a set bit in the mask.
